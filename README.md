## ✨ 프로젝트 설명

```sh
    서비스는 사용자가 본 웹툰/웹소설에 대해 별점 및 리뷰를 남기고  
    별점에 따라 나의 취향에 맞는 작품을 추천해주는 서비스입니다.  
    또한 매주 새롭게 올라오는 회차에 대한 라이브톡을 진행하여  
    독자 모두가 모여 이야기하며 작품에 대한 감상을 나눌 수 있습니다.  
```
